A) Installation - top

Follow the steps below to set up the Calendar:

  1. Upload 'calendar.php' file and 'calendar/' folder to your web.
  2. Once you have uploaded it, go to 'calendar/install.php'
  3. Finally you would need to fill the forms and your Calendar is ready!

B) HTML Structure - top

This theme is a fixed layout with a main container. The calendar is nested within a div with an id of "eCalendar". The general template structure is the same throughout the template. Here is the general structure.

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following in "calendar/css/style.css":

	#eCalendar a {
		color: #someColor;
	}

Else if you would like to edit the background color/image of the main theme, you would go to line 31 (to the body tag):

	body {
		background: #someColor/Image;
	}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

C) CSS Files and Structure - top

D) JavaScript - top

This theme imports three Javascript files.

  1. jQuery
  2. Bootstrap
  3. My custom script ('calendar/js/script.js')
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Most of the animation in this site is carried out from the Bootstrap script.
  3. The main file that powers the Calendar is the custom script.

E) PHP and MySQL - top

All the data is imported from the MySQL Database. In the admin folder you would find the settings file ('calendar/admin/settings.php') which contains:

  • The Database Connection (DB Host, DB Name, etc)
  • Your Admin Account to log into the Calendar's Control Panel
  • The Calendar style ('Modern' or 'Classic')
  • The Calendar first weekday ('Sunday' or 'Monday')

There are three PHP files in 'calendar/admin/php' listed below:

  1. config.php: the Control Panel Configuration
  2. func_events.php: the Control Panel functions
  3. headerAdmin.php: it's included when you are loged as the admin
